# Webplatform Frontend

Это приложение на React, созданное с помощью Vite и TypeScript.

## Настройка проекта

### Зависимости

Все зависимости проекта перечислены в файле `package.json`.

Чтобы установить их, перейдите в директорию `Webplatform` и выполните команду:

```bash
npm install
```

### Запуск сервера для разработки

Чтобы запустить локальный сервер для разработки, выполните команду:

```bash
npm run dev
```
